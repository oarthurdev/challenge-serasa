# Challenge Serasa API

API for my project to New Developers Program Serasa.

## Installation

Use the package manager [npm](https://www.npmjs.com/) to install this project.

```bash
npm install
```

## Structure and fill the database.

```zsh
npx knex migrate:latest
npx knex seed:run
```

## Start API

```zsh
npm start
```
